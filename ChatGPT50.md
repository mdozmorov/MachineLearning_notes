## Pro Tips

- Stack tokens for powerful prompts:  
  `ACT AS: Project Manager ‚Äî SWOT ‚Äî FORMAT AS: Table ‚Äî GUARDRAIL: Factual only`

- Use pipe symbols (`|`) to chain commands:  
  `SIMPLIFY | HUMANIZE | FORMAT AS: Bullet points`

  `CONTEXT: B2B SaaS startup | AUDIENCE: Investors | EXEC SUMMARY | FORMAT AS: Presentation slides`
